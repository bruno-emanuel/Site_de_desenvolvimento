Site institucional desenvolvido em HTML para a empresa Hi-Tech Solutions, do técnico em informática Ivan Leal.

## Sobre o projeto

Projeto criado com foco em:

- Visual moderno
- Layout responsivo
- Melhor experiência mobile
- Conversão via WhatsApp
- Design profissional
- Performance e usabilidade

O site apresenta os serviços de assistência técnica, manutenção de computadores, upgrades, redes e suporte especializado em Luziânia - GO.
